O que é Git e GitHub?

GitHub é uma plataforma feita para hospedagem de códigos de programação. Os objetivos principais são o controle e a colaboração, permitindo que usuários desenvolvam projetos juntos, estando em qualquer lugar do mundo.

O GitHub é uma plataforma de controle de versões gratuita que serve para gerir projetos e permite a colaboração entre uma grande comunidade de desenvolvedores, contribuindo com melhorias e comentários.

Configurando Chave SSH

Uma chave SSH é necessária para autenticar sua identidade ao interagir com repositórios remotos no GitHub. Siga os passos abaixo para configurar uma chave SSH:

  1. Verifique se você já possui uma chave SSH: 
    ls ~/.ssh/
  2. Se você não tiver uma chave SSH, crie uma: 
    ssh-keygen -t rsa -b 4096 -C "seu-email@example.com"
  3. Copie a chave pública para a área de transferência: 
    cat ~/.ssh/id_rsa.pub | clip (Windows) ou pbcopy (Mac)
  4. Cole a chave pública em suas configurações do GitHub: https://github.com/settings/keys

Criando e Clonando um Repositório

Para criar um novo repositório no GitHub e cloná-lo em seu computador, siga os passos abaixo:

  1. Entre na sua conta GitHub e clique em "New" (Novo) para criar um novo repositório.
  2. Preencha os detalhes do repositório, como nome, descrição e opções de visibilidade.
  3. Clique em "Create Repository" (Criar Repositório).
  4. Após criar o repositório, você verá a URL do repositório. Copie essa URL.
  5. No seu computador, abra o terminal e navegue até a pasta onde deseja clonar o repositório.
  6. Cole a URL do repositório e execute o seguinte comando para cloná-lo: 
    git clone URL_DO_REPOSITORIO
  7. Você agora tem uma cópia local do repositório para colocar em seu computador.
